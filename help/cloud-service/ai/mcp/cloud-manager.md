# Servidor MCP de Cloud Manager

El servidor MCP de Cloud Manager permite a los desarrolladores de AEM ejecutar canalizaciones, depurar errores y acceder a las herramientas de Cloud Manager directamente en su IDE, sin cambiar de contexto.

>[!AVAILABILITY]
>
> Cloud Manager MCP Server está ahora en versión beta. Para solicitar acceso, envíe un correo electrónico a [aemcs-mcp-feedback@adobe.com](mailto:aemcs-mcp-feedback@adobe.com) con una descripción de su caso de uso.
