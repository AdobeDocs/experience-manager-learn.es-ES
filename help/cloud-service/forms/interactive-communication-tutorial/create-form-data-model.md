# Crear modelo de datos de formulario para documento IC

Cree un modelo de datos de Forms para integrar fuentes de datos externas con comunicaciones interactivas en Adobe AEM. Este proceso implica la configuración de un servicio RESTful, la carga de un archivo Swagger y la configuración de puntos finales de servicio para recuperar y enlazar datos de forma dinámica. Aprenda a conectarse de forma segura a servicios externos y pruebe el modelo para garantizar una recuperación de datos correcta.

Se ha implementado un servidor API ficticio que simula el servicio Pedidos con fines de desarrollo y prueba. Expone un punto final para recuperar pedidos de un usuario determinado (por ejemplo, por ID de usuario) y devuelve datos de pedidos predefinidos o generados dinámicamente en el mismo esquema que la API de producción.

El archivo swagger utilizado en la creación del modelo de datos de formulario se puede [descargar desde aquí](assets/UsersAndOrders.json)
