# Crear tema

Una temática contiene detalles de estilo para componentes y paneles y se puede reutilizar en diferentes formularios. Los estilos incluyen propiedades como colores de fondo, colores de estado, transparencia, alineación y tamaño. Al aplicar la temática al formulario, el estilo especificado se reflejará en los componentes correspondientes del formulario.
