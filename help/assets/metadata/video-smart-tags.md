# Etiquetas inteligentes de vídeo

Experience Manager Assets aprovecha Adobe Sensei para etiquetar recursos de vídeo de forma inteligente con palabras clave que describen elementos clave del vídeo, lo que permite a los usuarios de AEM descubrirlos fácilmente mediante la búsqueda de palabras clave.
